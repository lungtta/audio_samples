🔊 LungTTA

Text-to-Audio Generation of Synthetic Lung Sounds for Respiratory Health

This demo showcases synthetic respiratory audio generated using LungTTA, a diffusion-based model designed for clinically relevant lung sound synthesis.

View Code on GitHub

Abstract

We introduce LungTTA, a text-to-audio generation framework for synthesizing clinically relevant respiratory sounds. Unlike general-purpose audio generation models, LungTTA incorporates domain-specific conditioning to generate coughs, breathing patterns, and vocal sounds aligned with clinical attributes such as age, smoking status, and respiratory conditions. The model is based on latent diffusion and is designed to address data scarcity in respiratory health applications. We evaluate LungTTA using objective metrics and human listening studies, demonstrating improved realism and clinical relevance compared to existing approaches.

Method Overview

LungTTA is a diffusion-based generative model conditioned on textual prompts describing patient attributes and respiratory characteristics. The system integrates domain-aware conditioning and structured prompts to guide the generation process toward clinically meaningful outputs. This enables controllable synthesis of respiratory sounds for downstream machine learning tasks and evaluation.
